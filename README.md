# 💸 Expense Tracker

A modern and minimal expense tracking web application built with **HTML, CSS, and JavaScript**.
This project allows users to manage daily expenses with categories, dates, and persistent storage.

---

## 🚀 Features

* Add new expenses with:

  * Name
  * Amount
  * Category
  * Date
* View total balance dynamically
* Delete expenses
* Data persistence using **localStorage**
* Clean and modern UI design
* Responsive and user-friendly layout

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox, modern UI styling)
* Vanilla JavaScript (DOM manipulation, localStorage)

---

## ⚙️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. Start adding your expenses

> No installation or dependencies required.
